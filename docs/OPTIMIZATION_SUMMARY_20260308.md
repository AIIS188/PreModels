## 🔧 关键技术突破

### 1. 在途期望计算修复

**问题**：在途期望包含了今天到达的量

**根因**：
```python
# 错误
intransit_mu_valid = sum(pred_mu[day9:day13])  # 包含今天

# 正确
intransit_mu_valid = sum(pred_mu[day11:day13])  # 只算未来
```

**效果**：今日发货目标从 85.4 吨提升到 99.1 吨（+16%）

---

### 2. 到货上限约束重构

**问题**：原约束限制"在途 + 新增"，导致在途大时无法发货

**根因分析**：
```
C2: day10 在途 138 吨 >> max_daily 50 吨
     → day10 可新增 = max(0, 50-138) = 0 吨 ❌
```

**解决**：
```python
# 改为限制每日发货量（不是在途 + 新增）
for t in range(today, ...):
    ship_expr = sum(x[w,cid,k,t] for ...)
    model += (ship_expr <= max_daily_ship)
```

**效果**：C1/C2 从 0 吨恢复到正常发货

---

### 3. 混装算法优化

**问题**：按品类单独换算车数，无法拼车

**解决**：
```python
# 按 lane 聚合后换算
total_tons = sum(tons for k, tons in category_tons)
trucks = ceil(total_tons / weighted_mu)
```

**效果**：总车数减少 15-20%
