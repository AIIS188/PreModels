## 前端使用示例

### JavaScript

```javascript
// 获取今日发货计划
async function getTodayPlan() {
  const response = await fetch('http://127.0.0.1:8001/api/v1/plan/10');
  const data = await response.json();
  
  console.log('=== 发货建议 ===');
  data.shipments.forEach((s, i) => {
    console.log(`建议 ${i + 1}:`);
    console.log(`  ${s.suggestion}`);
  });
  console.log(`\n${data.summary}`);
  
  return data;
}

// 使用示例
getTodayPlan();
```

## API 特性

### 1. 清晰的发货建议

每条发货计划都包含：
- **warehouse**: 仓库（如 W1）
- **receiver**: 收货方（如 R1）
- **cid**: 合同号（如 HT-2026-001）
- **category**: 品类（如 A）
- **tons**: 吨数（如 37.00 吨）
- **trucks**: 车数（如 2 车）
- **suggestion**: 完整的中文建议描述

### 2. 智能总结

`summary` 字段提供一句话总结：
- 包含所有发货路线
- 显示每条路线的吨数和车数
- 显示总计吨数和车数

### 3. 易于集成

- RESTful API 设计
- JSON 格式响应
- 支持 CORS
- 提供 Swagger 文档
